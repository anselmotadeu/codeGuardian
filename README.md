# CodeGuardian

## Visão Geral

CodeGuardian é uma ferramenta que identifica códigos ruins, sugere boas práticas e analisa várias linguagens de programação.

## Tecnologias Usadas

- **Front-end**: React
- **Back-end**: Node.js, Express
- **Banco de Dados**: MongoDB
